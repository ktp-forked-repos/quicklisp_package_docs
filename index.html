quicklisp_package_docs by vlionix

quicklisp_package_docs describes the different systems in quicklisp.

One of the hassles with Common Lisp circa 2011 is figuring out what different packages offer. Quicklisp has simplified the problem of actually getting that package. This project aims for a short-term - duct tape & baling wire - solution to the problem of package description. Long-term projects have been prototyped by various members of the Common Lisp community, but are not currently deployed.

Dependencies

Right now, the HTML generation is done by a Perl script. http://xkcd.com/224/

Install

Download. Make. Browse the html.
